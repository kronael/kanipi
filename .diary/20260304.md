## Ansible changes (core/ansible)

- Added `flavor: _rhias/happy/sloth` to kanipi service entries in `hel1v5` vars.
  Flavor makes systemd run `docker run kanipi-rhias` etc. (per-instance image),
  so `deploy kanipi_rhias` only affects rhias. Previously all three shared `kanipi:latest`.
- Renamed whisper image from `whisper` to `kanipi-whisper` in vars.
- Image tags on host: `kanipi-rhias`, `kanipi-happy`, `kanipi-sloth` all exist.
